# Quick Tweet Search Extension

Chrome Extension to quickly search Tweet by Twitter user

https://chrome.google.com/webstore/detail/quick-tweet-search/hdkkckfalkjiojdncobdbapmfecmlnbi

## Usage

**Select Twitter user and Input search words**

## Feature
-  Start by Shortcut
-  Lightweight user search
-  History of searched users

## Setup

```shell
npm install
```

## Build

```shell
npm run build
```

## Run on Dev Server

```shell
npm run dev
```
